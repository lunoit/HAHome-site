Getting started

The setup wizard walks you through picking your devices by category. Select the entities you want to control and organise them into rooms.

3

Control

Your home is live. Tap to toggle, long-press for detail controls, and watch real-time state changes as they happen.

No. HAHome is an independent, third-party app. It uses Home Assistant's open REST and WebSocket APIs but is not made by or affiliated with Nabu Casa or the Home Assistant project.

HAHome uses OAuth to log you in securely via your browser. After login, it maintains a WebSocket connection for real-time state updates and uses the Home Assistant REST API to send commands. Your credentials are stored in the iOS Keychain — never in a third-party server.
